# vyuka-junit
Demonstrace Java JUnit testů a integrace s Travis.

Projekt využívá [Maven](https://maven.apache.org/) k přeložení projektu a spuštění testů.

Přeložit a otestovat celý projekt můžete takto:

```shell
mvn test
```
